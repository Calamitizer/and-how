IPHM is a single-page webapp that visualizes global IPv6 geolocatio data in a interactive heatmap. As the map pans and zooms, the new visual latitude/longitude bounding box is calculated, the server is queried for IP data within it, and the client renders the heatmap.

IPHM also publicly exposes its API, returning a .json of IPv6 data within a given geographical bounding box.
