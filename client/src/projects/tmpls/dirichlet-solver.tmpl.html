Dirichlet Solver

Github

A MATLAB script for numerically solving the Laplace equation, with Dirichlet boundary conditions inside an irregular (non-rectangular) region. The Laplace equation is the centerpiece of steady-state heat transfer and potential theory.

The generated solution is displayed in a 3-D plot, which can be swung to an arbitrary viewing angle. (In MATLAB, that's the easy part.)

The irregular region is defined by specifying a binary inclusion function on the region's rectangular convex hill, and, as such, the script is particularly sensitive to resolution.

Written for Numerical Analysis II at NCSU.
